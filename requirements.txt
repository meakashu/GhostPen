# Contributing to GhostPen

We welcome contributions! To get started:

1. Fork the repository.
2. Clone your fork: `git clone https://github.com/your-username/GhostPen.git`
3. Create a new branch: `git checkout -b feature-branch`
4. Make your changes and commit them.
5. Push the changes: `git push origin feature-branch`
6. Open a Pull Request.

Please follow the coding standards and comment your code properly.
